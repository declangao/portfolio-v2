This is my personal portfolio site built with React, Next.js, TypeScript and Tailwind.

## Features

- Full-stack with backend REST API
- Sending emails with [Resend](https://resend.com)
- Multiple Framer Motion examples
- Light/dark theme

## Getting Started

Development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
